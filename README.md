## 🚀 **Quick Start**

### **Installation**
```bash
# Clone the repository
git clone https://github.com/CreatorOss/ultracrypto-framework.git
cd ultracrypto-framework

# Install dependencies
pip install -r requirements.txt

# Run demonstration
python demo/ultracrypto_demo.py
```

### **Basic Usage**
```python
from ultracrypto import UltraCryptoEngine

# Initialize the framework
engine = UltraCryptoEngine()

# Multi-dimensional encryption
encrypted = engine.encrypt("Your secret data", key="your-key", method="multi-layer")

# Decryption
decrypted = engine.decrypt(encrypted, key="your-key", method="multi-layer")

print(f"Original: Your secret data")
print(f"Decrypted: {decrypted}")
```
